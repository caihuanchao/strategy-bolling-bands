# 成交量分析策略

## 问题陈述

如何为量化策略工作台新增一个成交量分析策略，使其与现有布林带触碰和双均线交叉策略并列，整合量价背离、放量突破、OBV 能量潮、缩量回调、形态识别等核心量价分析能力，并通过策略注册中心无缝接入前端仪表盘？

## 推荐方向

**单一整合策略（Direction A）**：一个 `VolumeAnalysisStrategy(StrategyBase)` 覆盖所有子维度。

- `strategy_id = "volume_analysis"`，`strategy_name = "成交量分析"`
- `generate_signals()` 计算 OBV、量比、量价趋势，检测 4 类事件（背离/突破/回调/形态）
- 策略内部轻量共振：≥2 类事件同时触发 → 增强信号
- 新建 `volume_interpreter.py` 提供策略专属结构化中文解读
- 预设 3 组：标准、敏感（短线）、滞后（中长线）

## 需验证的关键假设

- [ ] OBV 纯价格方向累积计算，不需要额外数据源 —— 用现有 OHLCV 即可验证
- [ ] 量价背离检测 lookback 窗口设为 20 日，灵敏度可接受 —— 用历史数据回测确认
- [ ] 形态识别最低需要 10-20 个交易日数据 —— 检查自选股数据是否满足最小长度
- [ ] 轻量共振 ≥2/4 阈值在实盘中信号频率合理 —— 用扫描结果数量判断

## MVP 范围

| 包含 | 不包含 |
|------|--------|
| OBV 计算 + 趋势与背离检测 | Volume Profile（POC/VA，需价格维度聚合） |
| 量价背离（顶背离→卖出，底背离→买入） | VWAP（需要日内分时数据） |
| 放量突破（量比 1.5/2.0/3.0 三级） | MFI 指标（资金流量指数） |
| 缩量回调（趋势过滤 + 缩量确认） | 完整多维共振表（≥3 条件，后续迭代） |
| 4 种量价形态识别（吸筹/派发/高潮/整理） | OBV 均线平滑（MVP 用原始值） |
| 策略内部轻量共振（≥2/4 条件→增强信号） | |
| `volume_interpreter.py` 策略专属结构化解读 | |
| 3 组预设参数模板 | |

## 不做（及原因）

- **Volume Profile** — 需要按价格维度（纵轴）聚合成交量，与现有时间序列架构完全不同，工作量大
- **VWAP** — 需要日内分时数据，当前系统仅使用日线 OHLCV
- **MFI** — 与 RSI 高度相似但需要额外计算，边际价值有限；MVP 后用 RSI 作为辅助指标
- **完整 5 条件共振表** — 过于复杂，MVP 先验证轻量共振的效果
- **拆分为 5 个独立策略** — 注册中心膨胀，共振逻辑难以实现，前端切换负担重

## 待解决问题

- 量价背离的 lookback 窗口如何确定？（建议 20 日，需实际验证）
- 形态识别使用滑动窗口还是固定分段？（建议滑动窗口 + 回溯评分）
- 缩量回调是否需要 MA50/MA200 趋势过滤？（建议 MVP 用 MA20/MA50 简化版）
