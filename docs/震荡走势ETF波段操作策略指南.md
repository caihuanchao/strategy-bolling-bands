# 震荡走势 ETF 波段操作策略指南

---

## 一、先判断：你面对的是真震荡还是假震荡

震荡策略最大的风险是：你以为在震荡，其实趋势正在酝酿。进场前先确认三件事：

```
1. 价格是否在明确的水平区间内运行？（至少 3 次触及上下沿）
2. 均线是否走平、缠绕？（MA20/MA60 反复交叉，无明确斜率）
3. ATR 是否处于低位或下降中？（波动率收敛 = 震荡特征）
```

三者全满足，才是适合震荡策略的土壤。

---

## 二、核心策略：布林带 + RSI 均值回归

这是震荡市最经典的组合，逻辑互不重叠：

| 指标 | 角色 | 在震荡市中的信号 |
|------|------|-----------------|
| **布林带** | 价格边界 | 触及下轨 = 超卖区边界；触及上轨 = 超买区边界 |
| **RSI(14)** | 动能极端 | ≤ 30 超卖；≥ 70 超买 |

### 入场规则

```
买入：价格触及布林带下轨 + RSI ≤ 35 + 出现止跌K线形态（锤子线/看涨吞没）
卖出：价格触及布林带上轨 + RSI ≥ 65 + 出现滞涨K线形态（射击之星/看跌吞没）
```

### ETF 参数的特别调整

ETF 波动通常小于个股，布林带默认参数 (20, 2) 可能会让上下轨过窄。宽基 ETF 建议用 **(20, 2.5)** 或 **(20, 3)**，让触碰信号更有意义。

---

## 三、网格交易 — 最纯粹的震荡策略

当震荡区间清晰时，网格是最直接的手段：

```
确定箱体上下沿（至少 3 次确认）
    ↓
将区间等分为 5-8 格
    ↓
每格挂等额买单（下方）和卖单（上方）
    ↓
价格触及 → 自动成交 → 反向挂单
```

### ETF 做网格的独特优势

- 没有个股暴雷风险，越跌越买不会归零
- 费率低（管理费 + 托管费通常 0.2%-0.6%），频繁交易成本可控
- 流动性好，不存在挂单无法成交的问题

### 风险

如果震荡区间被突破，网格会持续朝一个方向建仓，可能重仓套牢。必须设置**止损线**，跌破箱体下沿 2%-3% 即暂停网格。

---

## 四、KDJ 低位金叉 / 高位死叉

KDJ 在震荡市的表现优于趋势市，因为它本身就是摆动指标：

```
J 值 < 0 后回升 + KDJ 金叉 → 买入
J 值 > 100 后回落 + KDJ 死叉 → 卖出
```

**注意**：KDJ 在日线很灵敏，假信号多。ETF 波动慢，建议用 **KDJ(9,3,3)** 的日线级别，并至少等待金叉/死叉后的**第二根 K 线确认**再动手，避免反复交叉的陷阱。

---

## 五、分时级别的辅助：分时均线 + MACD

用日线级别确定震荡区间后，在更短周期上精确择时：

```
15 分钟或 30 分钟图上：
  价格回踩分时均线 + MACD 零轴附近金叉 → 精确买入点
  价格远离分时均线 + MACD 高位死叉 → 精确卖出点
```

这个技巧可以改善波段策略的入场和出场精度，把日内 1%-2% 的滑点省下来。

---

## 六、ETF 波段操作的特殊考量

### 1. 品种选择

| ETF 类型 | 震荡特征 | 适合度 |
|----------|---------|--------|
| 宽基 ETF（沪深 300、中证 500） | 大周期震荡，区间宽 | 适合，空间够 |
| 行业 ETF（券商、白酒） | 阶段性震荡，突发趋势多 | 需警惕趋势突破 |
| 红利 ETF | 低波动，区间窄 | 空间有限，性价比低 |
| 债券 ETF | 极低波动 | 不适合波段 |
| 跨境 ETF（纳指、恒生） | 跳空多，区间不可靠 | 谨慎 |

### 2. 仓位管理

震荡策略的核心不是择时，是**仓位节奏**：

```
箱体下沿附近：仓位加到 70%-80%
箱体中轴附近：仓位维持 40%-50%
箱体上沿附近：仓位降到 20%-30%
```

永远不要满仓进、空仓出，因为震荡区间的边界随时可能被突破。

### 3. 手续费考量

ETF 交易免印花税（A 股 ETF），但佣金和滑点是实际损耗。假设每笔完整买卖成本 0.05%（万 2.5 佣金双向），一年做 20 个来回就是 1% 的固定损耗。波段空间至少要在 3% 以上才值得动手，否则赚的钱都在给券商打工。

---

## 七、策略失效的预警信号

当以下信号出现时，震荡策略要暂停，切换到趋势思维：

```
1. 布林带开口突然扩大（带宽 20 日最高）→ 可能突破在即
2. 价格放量突破箱体上沿或下沿 → 趋势启动
3. ADX(14) 从 20 以下升破 25 → 震荡转趋势的量化确认
4. 均线从缠绕转为多头/空头排列 → 趋势方向确立
```

**核心原则**：震荡策略吃的是区间内的均值回归，一旦区间本身不存在了，这套逻辑就失效了。不要跟已经启动的趋势对着干。

---

## 八、与其他策略的配合

| 场景 | 推荐策略 | 说明 |
|------|---------|------|
| 震荡确认阶段 | 本策略（布林带 + RSI + 网格） | 均值回归逻辑主导 |
| 震荡区间的日线精确入场 | MACD + RSI + 成交量三重确认 | 提高入场精度 |
| 突破后的趋势跟进 | 周线 RSI 策略 | 震荡转趋势时，切换至趋势策略 |
| 波动率监控 | KDJ + 布林带 + ATR 波动率 | ATR 收缩/扩张辅助判断 |

**核心原则**：震荡和趋势是同一个市场的两种状态。震荡策略赚的是区间回归的钱，趋势策略赚的是方向延续的钱。二者不能混用，但必须能平滑切换。
