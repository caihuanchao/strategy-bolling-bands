# MACD + RSI + 成交量三重确认策略深度指南

---

## 一、为什么是这三个指标：互补性分析

单一指标的致命弱点是**盲区**。三重确认的本质不是堆砌指标，而是让三个指标覆盖彼此的盲区。

| 指标 | 类型 | 核心能力 | 盲区 |
|------|------|----------|------|
| **MACD** | 趋势 + 动量 | 判断趋势方向和强度变化 | 震荡市频繁假信号 |
| **RSI** | 超买超卖 | 判断价格是否极端、是否背离 | 强趋势中持续钝化 |
| **成交量** | 资金验证 | 验证价格变动的真伪 | 不能独立判断方向 |

**三者如何互补**：

```
MACD 说 "趋势向上"  → 但不知道是不是尾声
RSI 说  "还没超买"   → 补上了：趋势还有空间
成交量说 "资金在跟进" → 再确认：信号是真金白银推出来的

三个指标从不同维度指向同一结论时，
本质是三套独立的信息源在交叉验证，胜率质变。
```

## 二、各指标在策略中的角色分工

### 1. MACD — 趋势裁判

**原始公式**：

```
MACD 线 = EMA12 - EMA26（快慢均线差值）
信号线 = EMA9(MACD 线)（MACD 的 9 日 EMA）
柱状图 = MACD 线 - 信号线
```

**在本策略中的三项核心职责**：

```
职责一：趋势方向判定
  MACD 线 > 0 且 MACD 线 > 信号线 → 多头趋势，只做多
  MACD 线 < 0 且 MACD 线 < 信号线 → 空头趋势，只做空
  二者矛盾（如 MACD > 0 但死叉）→ 震荡或转势，等待

职责二：动量变化检测
  柱状图由绿变红（由负变正）→ 多头动量开始增强
  柱状图高度连续放大 → 趋势加速
  柱状图连续缩短 → 趋势减速（警戒信号，不论方向）

职责三：背离检测（MACD 背离是所有背离中权重最高的）
  价格新高 + MACD 线未新高 → 顶背离 → 最可靠的顶部预警
  价格新低 + MACD 线未新低 → 底背离 → 最可靠的底部预警
  注意：MACD 背离 + RSI 背离 = 双重背离，强烈信号
```

**常见误区修正**：

- 金叉/死叉不是 MACD 的最强用法——**背离**才是
- 零轴附近的交叉比远离零轴的交叉更有意义
- MACD 柱状图的变化比金叉死叉更灵敏，适合提前预警

### 2. RSI — 极端探测器

**原始公式**：

```
RSI = 100 - [100 / (1 + 平均涨幅 / 平均跌幅)]
一般参数：14 周期
超买阈值：70（强势市场可用 80）
超卖阈值：30（弱势市场可用 20）
```

**在本策略中的三项核心职责**：

```
职责一：过滤追高风险
  MACD 金叉 + 成交量放量 → 看起来可以做多
  但如果 RSI > 75 → 价格已经拉得很高了
  → 不追！等 RSI 回落到 50-60 再进，或者直接放弃

职责二：捕捉背离信号（RSI 背离比 MACD 背离更敏感）
  价格新高 + RSI 未新高 → 顶背离
  价格新低 + RSI 未新低 → 底背离
  与 MACD 背离叠加时，确认度极高

职责三：趋势强度诊断
  RSI 持续在 40-80 区间运行 → 健康上升趋势，持仓
  RSI 持续在 20-60 区间运行 → 健康下降趋势，持币/持空
  RSI 频繁穿越 50 线 → 无趋势震荡，该策略暂停
```

**关键操作纪律**：

> 在 MACD 发出做多信号时，只要 RSI > 80，一律不进场。宁可错过这一波，也不在极端超买位接盘。

### 3. 成交量 — 真相验证器

**在本策略中的两项核心职责**：

```
职责一：确认信号有效性
  MACD 金叉当天，成交量不低于 20 日均量 → 信号有效
  MACD 金叉当天，成交量极度萎缩 → 信号存疑，跳过或减仓

职责二：排除顶底陷阱
  MACD 底背离 + 价格反弹 → 如果反弹不带量 → 假底，不参与
  MACD 顶背离 + 价格回落 → 如果回落缩量 → 真顶概率低，可能是洗盘
```

## 三、信号分级体系

### C 级信号：仅 MACD 交叉

```
条件：MACD 金叉（做多）/ MACD 死叉（做空）
可靠性：约 40-45%
处理：不交易，仅加入观察列表
```

### B 级信号：MACD 交叉 + RSI 配合

```
做多条件：
  ✅ MACD 金叉
  ✅ RSI 位于 30-65 区间（未超买）
  ✅ RSI 方向向上
可靠性：约 50-55%
处理：轻仓试探（正常仓位 50%），止损严格
```

### A 级信号：MACD 交叉 + RSI 配合 + 成交量确认（标准入场）

```
做多条件：
  ✅ MACD 金叉（零轴下方金叉更佳，空间更大）
  ✅ RSI 位于 35-60 区间
  ✅ RSI 方向向上
  ✅ 金叉当日成交量 ≥ 20 日均量的 1.0 倍
  ✅ 最近 3 天呈"缩量止跌 + 今日放量起涨"格局
可靠性：约 60-65%
处理：正常仓位，标准止损
```

### S 级信号：三重共振（高确信信号）

```
做多条件（必须全部满足）：
  ✅ MACD 底背离（价格新低 + MACD 未新低）
  ✅ RSI 底背离（价格新低 + RSI 未新低，且 RSI < 40）
  ✅ 放量阳线确认背离（成交量 > 20日均量的 1.5 倍）
  ✅ MACD 金叉确认背离有效
可靠性：约 70-75%（在有趋势过滤的情况下）
处理：正常仓位或略大仓位（风险不超过 3%），止损在背离低点下方
```

## 四、完整的进出场框架

### 做多流程

```
【Step 1 — 趋势过滤】
  价格 > MA50 且 MA50 方向向上 → 允许做多
  MA50 方向向下 → 只做反弹，不做趋势多单（降级处理）

【Step 2 — 信号等待】
  观察出现 A 级或 S 级信号的标的
  等待信号日的收盘价确认（不是盘中）
  记录以下数值：RSI 读数、MACD 柱高度变化、当日量比

【Step 3 — 入场执行】
  信号确认后，次日开盘入场（集合竞价后）
  仓位 = 账户风险金额 / (入场价 - 止损价)
  单笔风险不超过总账户的 2%

【Step 4 — 止损设置】
  初始止损：信号当日最低价下方 1 个 ATR
  或：近期波段低点下方

【Step 5 — 持仓管理】
  每根 K 线收盘后检查：
    □ MACD 柱状图是否仍在扩大（趋势健康）
    □ RSI 是否仍在 40-80 健康区间
    □ 成交量是否维持正常水平（没有突然萎缩）
    □ 以上三条全部正常 → 持仓
    □ 任何一条恶化 → 第二天关注，两条恶化 → 减仓

【Step 6 — 出场触发】
  以下任一条件触发平仓：
    1. MACD 死叉（优先级别最高）
    2. RSI 顶背离（价格新高 + RSI 未新高）
    3. RSI 进入超买区后回落（> 75 后下穿 75）
    4. 价涨量缩连续出现 3 根 K 线
    5. 价格跌破 MA20（辅助出场信号）
```

### 做空流程（镜像逻辑）

```
【Step 1 — 趋势过滤】
  价格 < MA50 且 MA50 方向向下 → 允许做空

【Step 2 — 信号等待】
  MACD 死叉 + RSI 位于 45-70 区间 + 放量阴线确认

【Step 3 — 入场】
  次日开盘入场，止损在信号日最高价上方 1 个 ATR

【Step 4 — 持仓与出场】
  镜像做多逻辑：MACD 金叉出场，RSI 底背离警觉，缩量反弹持有
```

## 五、实战中最常见的五种失败模式及对策

| 失败模式 | 原因 | 对策 |
|----------|------|------|
| **震荡市中反复止损** | MACD + RSI 在窄幅震荡中频繁交叉 | 加入 ADX 过滤，ADX < 20 时无视所有信号 |
| **RSI 钝化错过大行情** | 强趋势中 RSI 在 80+ 持续数周 | 强趋势中提高 RSI 超买阈值到 85，或参考 RSI 趋势线突破 |
| **放量假突破** | 有量无价——放量后价格未跟进 | 不盘中追入，等收盘价确认；次日开盘价需高于信号日收盘价 |
| **背离后继续背离** | 背离后价格继续原方向运行 | 背离不是入场信号，是预警信号。等 MACD 交叉确认背离后才入场 |
| **过早止盈** | MACD 柱收窄但趋势未反转 | 分两批出场：MACD 收窄时出 1/3，死叉时出 2/3 |

## 六、多时间周期验证

这是本策略最能提升胜率的进阶用法。

```
日线级别的三重确认信号 + 周线级别的方向确认 = 高确信

做多决策矩阵：

  周线 MACD > 0 且日线 A/S 级信号 → 满分做多（最理想）
  周线 MACD > 0 但日线仅 B 级    → 可轻仓试探
  周线 MACD < 0 但日线 S 级      → 仅做反弹，目标不宜过高
  周线 MACD < 0 且日线仅 B 级    → 不参与

核心原则：周线定战略（做不做），日线定战术（怎么做）
```

## 七、推荐参数配置

```
核心指标：
  MACD：EMA12 / EMA26 / Signal 9（经典参数，无需修改）
  RSI：14 周期
  成交量均线：20 日

辅助过滤：
  趋势均线：MA50（EMA 或 SMA 均可）
  ADX：14 周期，阈值 20

信号优先级：
  S 级（三重共振）> A 级（标准确认）> B 级（试探）> C 级（仅观察）

仓位：
  S 级：风险 2-2.5%   A 级：风险 1.5-2%   B 级：风险 0.5-1%
```

## 八、总结

MACD + RSI + 成交量的组合之所以经典，在于三个指标的信息源几乎完全独立：

- MACD 看的是**均线的距离和变化速度**（趋势动量）
- RSI 看的是**涨跌幅度比的归一化值**（极端程度）
- 成交量看的是**市场参与者的资金规模**（认可程度）

当它们在同一个位置、指向同一个方向时，意味着三套独立的"传感器"都检测到了同样的信号——出错概率不是相加，而是相乘式下降。这正是三重确认策略的核心价值。
