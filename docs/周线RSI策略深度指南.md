# 周线 RSI 策略深度指南

---

## 一、为什么是"周线"？

在所有的 RSI 时间框架中，**周线 RSI 是噪音比最低、信号质量最高的周期之一**。原因：

| 周期 | 噪音水平 | 信号频率 | 假突破概率 |
|------|---------|---------|-----------|
| 日线 | 高 | 频繁 | 较高 |
| **周线** | 低 | 适中 | 低 |
| 月线 | 极低 | 稀少 | 极低 |

周线用 5 根日线合成一根 K 线，天然过滤了日内波动和消息面噪音——它反映的是**资金的真实态度**，不是情绪。

## 二、经典策略框架

### 1. 超买超卖 — 但参数要调整

传统日线 RSI 用 30/70 分界，但**周线 RSI 永远达不到日线的极端值**：

```
日线 RSI 范围：常见 15-85
周线 RSI 范围：常见 25-75（极少跌破 20 或突破 80）
```

所以周线 RSI 的实战参数调整为：

| 区域 | 日线阈值 | 周线阈值 |
|------|---------|---------|
| 超卖区 | ≤ 30 | ≤ 35 |
| 超买区 | ≥ 70 | ≥ 65 |
| 极端超卖 | ≤ 20 | ≤ 30（多年大底） |

**关键认知**：周线 RSI 从 35 以下的超卖区向上突破 50 中轴，是比较可靠的趋势反转信号。反之，从 65 以上跌破 50，意味着中期趋势转弱。

### 2. RSI 背离 — 周线上最有效的信号

这是周线 RSI 策略中胜率最高的一种模式。

**底背离（看涨）**：

```
价格创新低 → 但周线 RSI 不创新低
含义：下跌动能衰竭，聪明资金在吸筹
级别：通常对应 3-6 个月的中期底部
```

**顶背离（看跌）**：

```
价格创新高 → 但周线 RSI 不创新高
含义：上涨动能衰竭，主力在出货
级别：通常对应 2-4 个月的中期顶部
```

**日线 vs 周线背离的关键差异**：

日线背离可能反复出现 3-4 次才真正反转，但周线背离通常出现 1-2 次就会生效。

### 3. RSI 趋势线突破

在周线 RSI 图表上画趋势线，效果远好于价格图上的趋势线：

- RSI 本身是一个**领先指标**，它的趋势线突破通常比价格趋势线突破**早 1-3 周**
- RSI 的下降趋势线被突破 = 动能在趋势反转前先转向
- 这条规则尤其适用于判断长期横盘后的突破方向

### 4. RSI 中轴（50）策略

RSI = 50 是牛熊分界线。在周线级别，这个信号很重：

```
周线 RSI 持续在 50 以上 → 中期牛市，回调即买入机会
周线 RSI 持续在 50 以下 → 中期熊市，反弹即卖出机会
周线 RSI 从下方突破 50 并站稳 2 周以上 → 中期上涨行情启动信号
```

## 三、实战组合策略

将以上规则整合为一个完整的周线 RSI 策略：

### 买入条件（需同时满足）

1. 周线 RSI 从 35 以下回升至 35 以上（超卖反弹）
2. 周线 RSI 出现底背离形态
3. 周线 RSI 突破自身下降趋势线

### 卖出条件（任意一条触发）

1. 周线 RSI 达到 65 以上后回落
2. 周线 RSI 出现顶背离
3. 周线 RSI 跌破 50 中轴

### 策略特性评估

| 维度 | 评价 |
|------|------|
| 信号频率 | 一年通常 2-3 次，不会过度交易 |
| 持仓周期 | 单次 4-12 周，适合有耐心的中长线资金 |
| 信号确认 | 周线收盘价确认，避免盘中假突破 |
| 入场精度 | 日线级别可能存在更优具体买点（可用日线 RSI 辅助） |
| 震荡市表现 | 信号偏弱偏多，需配合趋势过滤 |

## 四、风险与边界条件

### 1. 不要用周线 RSI 做短线

周线信号的确认需要等到周五收盘，中途的波动不代表信号成立。

### 2. RSI 在强势趋势中的钝化

大牛市中 RSI 可能长期维持在 60-75，用超买去卖会被轧空。应对原则：

```
趋势市 → 放宽超买阈值，收紧超卖阈值
震荡市 → 恢复标准阈值
```

### 3. 背离不是立即反转

周线背离出现后，价格可能还要惯性运行 1-3 周才真正转向。不要看到背离就立刻重仓进场。

### 4. 成交量配合原则

- RSI 底背离 + 缩量止跌 → 确认度更高
- RSI 顶背离 + 放量滞涨 → 确认度更高
- 没有量配合的背离 → 保持警惕，可能还有反复

### 5. 参数敏感性

标准 RSI 参数为 14 周期（周线即 14 周）。缩短参数（如 9）会增加信号频率但降低可靠性；延长参数（如 21）会减少信号但提高质量。建议以 14 为基础，根据品种波动特性微调。
