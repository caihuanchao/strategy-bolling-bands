# 双均线交叉策略完整指南

## 一、策略原理

双均线交叉（Dual Moving Average Crossover）是最经典的趋势跟踪策略之一。**核心思想**：通过两条不同周期的移动平均线的相对位置关系，判断市场趋势方向并生成交易信号。

## 二、均线指标深度解读

### 两条均线的角色分工

| 均线 | 周期 | 角色 | 特征 |
|------|------|------|------|
| **快线（短期均线）** | 5/10/20 日 | 价格变化的"先遣部队" | 敏感、滞后小，但噪声多 |
| **慢线（长期均线）** | 50/60/200 日 | 趋势的"主心骨" | 稳定、过滤噪音，但滞后大 |

### 常用周期组合

| 组合 | 适用场景 | 特点 |
|------|----------|------|
| **MA5 + MA20** | 短线交易 | 信号多，假信号也多 |
| **MA10 + MA50** | 中短期波段 | 平衡型，最常用的默认组合 |
| **MA20 + MA60** | 中线趋势 | 过滤噪音，趋势确认更可靠 |
| **MA50 + MA200** | 长线趋势（金叉/死叉经典组合） | 信号少但质量高，机构常用 |

### 关键概念：金叉与死叉

```
金叉 (Golden Cross)：快线从下向上穿越慢线 → 看涨信号
死叉 (Death Cross)：快线从上向下穿越慢线 → 看跌信号
```

## 三、买入/卖出信号详解

### 信号分级（按可靠性从低到高）

#### 1. 弱势信号 — 仅快慢线交叉

```
条件：快线上穿慢线（做多）/ 快线下穿慢线（做空）
可靠性：★★☆☆☆
风险：假信号多，尤其在震荡市中容易被反复打脸
```

#### 2. 标准信号 — 交叉 + 均线方向确认

```
买入条件（三者同时满足）：
  ✅ 快线上穿慢线
  ✅ 快线方向向上（斜率 > 0）
  ✅ 慢线方向向上或走平

卖出条件（三者同时满足）：
  ❌ 快线下穿慢线
  ❌ 快线方向向下（斜率 < 0）
  ❌ 慢线方向向下或走平
可靠性：★★★☆☆
```

#### 3. 强势信号 — 交叉 + 均线方向 + 价格位置

```
强势做多信号：
  ✅ 快线上穿慢线
  ✅ 快线 > 慢线 > 价格（三条线多头排列）
  ✅ 价格在均线上方运行
  ✅ 成交量放大配合

强势做空信号：
  ❌ 快线下穿慢线
  ❌ 快线 < 慢线 < 价格（三条线空头排列）
  ❌ 价格在均线下方运行
  ✅ 成交量放大配合
可靠性：★★★★☆
```

#### 4. 金叉/死叉信号 — 最可靠的长线信号

```
金叉（50日上穿200日）：
  - 确认长期趋势转多
  - 通常在价格已上涨 20-30% 后才出现
  - 信号极少但质量极高
  - 历史上标普500的金叉后12个月胜率超 70%

死叉（50日下穿200日）：
  - 确认长期趋势转空
  - 熊市确认信号
  - 可能出现 10-15% 的滞后
可靠性：★★★★★（长线），滞后大是代价
```

## 四、进出场规则

### 入场

```
做多入场：
  1. 确认金叉信号出现
  2. 等待一根K线收盘价确认（避免盘中假突破）
  3. 止损设在近期低点下方或ATR*2的距离
  4. 仓位 = 账户风险金额 / 止损距离

做空入场：
  1. 确认死叉信号出现
  2. 等待一根K线收盘价确认
  3. 止损设在近期高点上方或ATR*2的距离
```

### 出场

```
出场规则（按优先级）：

1. 止损出场 — 价格触及预设止损位
2. 信号反转 — 出现相反方向的交叉信号
3. 均线走平 — 慢线斜率趋于零（趋势衰竭）
4. 价格远离均线 — 乖离率过大（超过历史标准差2倍）
5. 跟踪止损 — 沿慢线移动止损位（保护利润）
```

## 五、策略优化的关键技巧

### 1. 过滤假信号（震荡市杀手）

```
过滤方法：
- 加入 ADX 指标：ADX < 20 时忽略所有交叉信号
- 乖离率过滤：金叉时乖离率不宜过大（避免追高）
- 成交量确认：缩量交叉往往是假信号
- 多周期验证：周线趋势与日线信号同向时进场
```

### 2. 避免追涨杀跌

```
错误做法：裸交叉就进场
正确做法：
  - 金叉后等价格回踩慢线再进（更好的入场点）
  - 回踩时缩量止跌确认支撑
  - 这样入场价格更好，止损更近，盈亏比更高
```

### 3. 参数优化原则

```
不要过度优化：
  - 不要用"最佳参数"回测后直接实盘
  - 用滚动窗口检验参数稳定性
  - 好的参数应该在多个时间段都表现一致
  - MA10/50 这样的简单参数通常比优化结果更稳健
```

## 六、策略局限性

| 弱点 | 说明 | 应对方法 |
|------|------|----------|
| **震荡市巨亏** | 频繁假突破导致连续止损 | ADX 过滤 + 暂停交易 |
| **信号滞后** | 趋势已走大半才确认 | 配合动量指标提前预警 |
| **V型反转** | 均线来不及反应 | 必须有硬止损 |
| **假突破** | 盘中突破尾盘返回 | 以收盘价为准 |

## 七、实战框架示例

```
策略配置：
  快线：MA10
  慢线：MA50
  过滤器：ADX > 25
  仓位：每笔风险 ≤ 总账户 2%
  止损：2倍 ATR（14周期）
  出场：反向交叉信号 OR 跟踪止损（MA20）

做多流程：
  1. ADX > 25 → 允许交易
  2. MA10 上穿 MA50 → 金叉信号
  3. 等收盘价确认 → 次日开盘入场
  4. 止损 = 入场价 - 2*ATR
  5. 每日移动止损沿 MA20 跟踪
  6. MA10 下穿 MA50 → 平仓
```
